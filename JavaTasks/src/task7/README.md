# Задача 7 (RU)

**Описание:**  
Заведите три переменные типа `String`. С клавиатуры введите в них вашу фамилию, имя и отчество. Выведите в консоль ваше ФИО в одну строку. Реализуйте это двумя способами:
1) Используя `System.out.println`
2) Используя `System.out.printf`

# Task 7 (EN)

**Description:**  
Create three `String` variables. Enter your surname, first name, and middle name from the keyboard into them. Print your full name in one line to the console. Implement this in two ways:
1) Using `System.out.println`
2) Using `System.out.printf`
